# Projet - Création du front pour le projet backjs

## Objectif du projet

Ce projet a pour objectif de créer le front du projet backjs permettant de générer des quotes.
Nous avons créé une page html permettant d'afficher les citations, un générateur IA de citation, créer, afficher, modifier, supprimer des citations. Il y a également un moyen d'authentification et d'inscription afin d'avoir la possibilité d'avoir accès aux fonctions de modifications ou de création.
